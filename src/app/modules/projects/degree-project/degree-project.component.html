Remote control system in a wireless sensor network for monitoring cloud services within an industry 4.0 laboratory

The term industry 4.0 is widely used in Europe, it is also common to refer to this concept with terms such as "Smart Factory" or "Industrial Internet". In short, it is about the application to industry of the "Internet of Things" (IoT) model. All these terms have in common the recognition that manufacturing processes are in a process of digital transformation.
El monitoreo del sistema de telecontrol para la transmision de datos mediante los servicios en la nube de las variables fisicas medibles de los sensores, se llévo acabo en la ciudad de Coatzacoalcos Veracruz en el Instituto Tecnologico Superior de Coatzacoalcos, en el cual se tomaron las mediciones de temperatura ambiente, humedad ambiente y posicionamiento global, en el mes de octubre del 2019.

En conclusión, con respecto al análisis realizado a lo largo del desarrollo del proyecto, con el diseño propio y la plataforma de servicio en la nube gratuita se redujo el costo de implementación del sistema de telecontrol para el laboratorio de industria 4.0 para el Instituto Tecnológico Superior de Coatzacoalcos, obteniendo mediciones eficientes para dicha aplicación con la tecnología utilizada en el proyecto propuesto.
Para el sistema de telecontrol IoT bajo la red LPWAN, los sensores máximos que se pueden colocar son 6, solo si se reduce la resolución de lectura de cada sensor a un máximo de 2 bytes por sensor, en esta ocasión solo fueron necesarios 2 sensores con los cuales se leyeron 5 variables, ocupando la carga útil máxima que son 12 bytes de transmisión de datos para la red Sigfox, el consumo de energía eléctrica del módulo para la red LPWAN es de 304.5 mAh, la ventaja de la tecnología Sigfox es que al transmitir los datos se guardan en la nube de Sigfox sin necesidad de estar conectado al internet para transmitir esos datos, para visualizar los datos se puede hacer por medio de callbacks provenientes de cualquier plataforma web dedicada al monitoreo de variables físicas medibles, de esta forma se pueden visualizar desde cualquier red como la 5G, 4G, y IEE 802.11 a/b/n, obteniendo una amplia gama de redes para combinar con la tecnología Sigfox. En el caso del sistema de telecontrol IoT bajo la red WI-FI no se encuentran restricciones en cuanto al número de datos de transmisión, la transmisión de datos depende del paquete de internet con el que cuente, el consumo de energía eléctrica del módulo para la red IEEE 802.11 es de 131.5 mAh.
